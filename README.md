# PC Performance Lab

Full-stack dashboard for tracking PC hardware snapshots, tuning profiles and gaming performance sessions.

## Planned project

- Backend API: Java 21 + Spring Boot
- Frontend dashboard: Next.js + TypeScript
- Local agent: Java hardware scanner
- Docs: API notes, benchmark formats and project decisions

## Goal

The goal of this project is to track PC configurations, store tuning snapshots and compare benchmark/game performance sessions to understand whether a change actually improves performance.

## Status

Project created. Backend API will be implemented.
